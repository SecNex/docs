# Server

The server is a services that surf the web and take screenshots of the content. It is a headless browser that can be controlled via a REST API. We use Python, Playwright and Flask in combination with SocketIO to create a server that can be controlled remotely.

## Requirements

- Python >= 3.8
- Playwright
- Flask
- Flask-SocketIO

## Installation

```bash copy
git clone https://github.com/SecNex/edgesight-headless-server-python.git
cd edgesight-headless-server-python
pip install -r requirements.txt
```

## Usage

```bash copy
python index.py
```
