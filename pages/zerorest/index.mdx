# ZeroRest

ZeroRest is a simple, lightweight, and easy-to-use RESTful API framework for Python. It is designed to be fast, flexible, and easy to use, making it ideal for building web applications and services that require a RESTful API.

<a href="https://github.com/SecNex/zerorest" style="border: 1px solid #000; padding: 10px; border-radius: 5px; text-decoration: none; color: #000;">
    Get ZeroRest on GitHub
</a>

## APIs

- Microsoft Graph API
- Microsoft SharePoint API
- Microsoft Teams API

## Features

- Secure authentication and authorization
    - Delegate authentication
    - Application authentication
- Easy-to-use API client
- Built-in support for common API operations
